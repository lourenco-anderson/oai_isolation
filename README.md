# OAI ISOLATION

## Getting started

Clone this repo. Run `git submodule init` then follow the steps [here](https://gitlab.eurecom.fr/oai/openairinterface5g/-/blob/develop/doc/NR_SA_Tutorial_OAI_nrUE.md).

Build the code:
```bash
cmake -B build
cd build
make
```
